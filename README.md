# 📚 Biblioteca Escolar - Semana 15

## Descripción

Sistema web desarrollado para la Biblioteca Escolar **"Saber Digital"** utilizando HTML5, CSS3, JavaScript (ES6+) y Bootstrap 5. El proyecto implementa una interfaz moderna, responsiva e interactiva para la gestión de libros, permitiendo administrar el catálogo escolar de manera eficiente y accesible desde cualquier dispositivo.

Además, el sistema ha sido preparado para su publicación en Internet mediante **GitHub Pages**, permitiendo el acceso remoto para estudiantes y docentes.

---

## Características

* 📖 Visualización del catálogo de libros.
* ➕ Registro de nuevos libros mediante formularios y ventanas modales.
* 🚫 Validación para evitar registros incompletos.
* 🏷️ Clasificación por categorías.
* 🔍 Búsqueda y filtrado de libros.
* 📊 Estadísticas de préstamos y lecturas.
* 🏆 Reporte de libros más leídos.
* 🌙 Tema oscuro y claro.
* 💬 Tooltips y componentes interactivos de Bootstrap.
* ♿ Accesibilidad mediante atributos ARIA.
* 📱 Diseño responsivo para dispositivos móviles, tablets y escritorio.
* ☁️ Publicación en GitHub Pages.
